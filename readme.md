# FakeVoice - AI语音转换App

一个基于Silicon Flow API的Android语音转换应用。

## 功能特性

- 文字转语音
- 语音录制
- 语音变声
- 语音管理(上传、删除、播放)

## 技术栈

- Kotlin
- Jetpack Compose
- Hilt依赖注入
- Coroutines & Flow
- Retrofit2
- ExoPlayer
- Clean Architecture

## 开始使用

1. Clone项目
2. 在ApiConfig.kt中配置你的API密钥
3. 运行项目

## 开发环境要求

- Android Studio Hedgehog | 2023.1.1
- Kotlin 1.9.0
- Gradle 8.0
- minSdk 24
- targetSdk 34
